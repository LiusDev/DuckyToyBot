### JSClub Codefest SDK
JSClub Codefest SDK là SDK (Software Development Kit) - bộ công cụ hỗ trợ cho các thí sinh tham gia Codefest.

Các chức năng chính:
- Kết nối với Server thi đấu.
- Cung cấp các util xử lý dữ liệu.
- Cũng cấp thuật toán tìm đường (A* Search).
